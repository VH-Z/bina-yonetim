# 🏢 Bina Yönetim Sistemi

Python/Tkinter ile geliştirilmiş koyu temalı bina yönetim uygulaması.

## İndirme

👉 **[Releases](../../releases/latest)** sekmesinden son sürümü indirin.

- `BinaYonetim_Kurulum.exe` → Kurulum sihirbazı (önerilen)
- `BinaYonetim.exe` → Kurulum gerektirmez, taşınabilir

## Özellikler

- Daire ve sakin yönetimi
- Aidat takibi ve makbuz oluşturma
- Gelir / gider kayıtları
- Alacaklı ve ortak gider takibi
- Detaylı raporlar ve yazdırma
- Uyarı yazısı (PDF/Word)
- Koyu tema arayüz
